# 编译容器镜像等常用Docker命令

镜像上传

```
docker tag container-registry.oracle.com/database/enterprise:21.3.0.0 docker.io/wilburice/oracledb:21.3.0.0
docker push docker.io/wilburice/oracledb:21.3.0.0
docker tag container-registry.oracle.com/middleware/weblogic:14.1.1.0 docker.io/wilburice/weblogic:14.1.1.0
docker push docker.io/wilburice/weblogic:14.1.1.0
```

镜像导入导出

```
#导出多个镜像到一个文件中
docker save -o oracledb21c_weblogic.tar container-registry.oracle.com/database/enterprise:21.3.0.0 container-registry.oracle.com/middleware/weblogic:14.1.1.0
#导入
docker load < oracledb21c_weblogic.tar
```
